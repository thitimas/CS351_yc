Pseudocode

function Dijkstra(G, source): for each vertex v in G: dist[v] ← ∞; prev[v] ← null dist[source] ← 0 PQ ← priority queue; insert (0, source) while PQ is not empty: (d, u) ← PQ.extract-min() if d > dist[u]: continue // stale entry for each neighbor v of u with weight w(u,v): if dist[u] + w < dist[v]: dist[v] ← dist[u] + w; prev[v] ← u PQ.insert((dist[v], v)) // relax edge return dist, prev
