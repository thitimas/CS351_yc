Pseudocode

function KNN-Classify(training_set, x_query, k): distances ← [] for each (x_i, label_i) in training_set: d ← distance(x_query, x_i) distances.append((d, label_i)) sort distances ascending by d neighbors ← first k of distances votes ← count_labels(neighbors) return argmax(votes) // majority class
No "training" step — KNN is a lazy learner: it just stores the data and does all the work at query time.

Runtime Complexity

PhaseTimeSpace
Train (just store) O(1) O(n·d)
Predict (this demo) O(n·d + n·log n) O(n)
n = training size, d = feature dimensions. Computing all distances is O(n·d); sorting them to pick the K nearest is O(n·log n).
